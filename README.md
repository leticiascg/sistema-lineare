# Projeto Lineare

Sistema de gestão de tarefas desenvolvido para a disciplina de Fábrica de Software - Univille 2026.

## Sobre o Projeto

O Lineare é uma aplicação web de gestão de fluxos de trabalho focada na centralização de demandas pessoais. O sistema prioriza o ciclo essencial da tarefa: cadastramento, monitoramento visual e finalização efetiva.

## Tecnologias

- React 
- Vite
- CSS3

## Instalação

1. Clone o repositório:
```bash
link: https://github.com/leticiascg/sistema-lineare.git
git clone https://github.com/SEU_USUARIO/sistema-lineare.git
cd projeto-lineare
```

2. Instale as dependências:
```bash
npm install
```

3. Execute o projeto:
```bash
npm run dev
```

## Funcionalidades

- Criar novas tarefas
- Listar tarefas ativas e concluídas
- Marcar tarefas como concluídas
- Remover tarefas
